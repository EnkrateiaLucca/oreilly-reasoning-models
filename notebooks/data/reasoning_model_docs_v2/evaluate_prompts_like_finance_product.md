# Lesson 6 — Evaluate Prompts Like a Finance Product (Small Test Sets)

**Why it matters:** What isn’t measured won’t improve.

**Core ideas**

* Define **task metrics:** accuracy (exact match / tolerance bands), completeness, citation coverage, time saved.
* **Gold mini-sets (5–20 cases)** → cheap, repeatable checks before rollout.
* A/B testing prompts: baseline vs. improved; track **cost, latency, error rate**.

**Hands-on lab**

* Create a **10-item mini test set** for vendor categorization (e.g., SaaS, contractors, office).
* Compare two prompts; record **precision/recall**, and write a 5-bullet **decision note**.
* Deliverable: Test sheet + decision note + chosen prompt.

**Quality bar**

* Clear pass/fail rules; chosen prompt outperforms baseline on at least two metrics.
