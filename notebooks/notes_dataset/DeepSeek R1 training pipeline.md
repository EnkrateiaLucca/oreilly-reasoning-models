# DeepSeek R1 training pipeline

The full reasoning model (vs the proof-of-concept [[DeepSeek R1-Zero]]), built to fix R1-Zero's readability/language-mixing. **Five stages** (DeepSeek-AI, 2025):

1. **Pretrain → V3-Base** (MoE + MLA, ~671B total / ~37B active).
2. **Small-scale cold-start SFT.** A few thousand long chains-of-thought generated by R1-Zero and **rewritten by humans** for clean formatting and language consistency — an anchor before RL.
3. **GRPO on reasoning data** (≈ the R1-Zero RL process) with reward = **formatting + accuracy + language-consistency**. The language-consistency reward = ratio of tokens in the target language in the chain, added to stop language mixing.
4. **Large-scale SFT** mixing reasoning + non-reasoning, ~**3:1**:
   - **~600k reasoning pairs** (math/code/logic), produced by **rejection sampling** of "R1-so-far" outputs filtered by rules + a V3 LLM judge (keep good, drop bad).
   - **~200k non-reasoning pairs**, mostly reusing V3's SFT data (broad general domains).
5. **Final GRPO** on reasoning + non-reasoning. Reasoning → formatting + accuracy reward; general → **helpfulness + harmlessness** reward (harmlessness applied over the *whole* output incl. the think section). → **R1**.

**Result:** competitive with closed-source reasoning models (o1) — e.g. MATH-500 97.3, AIME-2024 79.8, Codeforces ~2029.
